# 이미 설치한 사용자 — 업데이트는 어떻게 되나요?

5240 PcOff Agent를 이미 PC/Mac에 설치해 두었다면, **새 버전이 나왔을 때** 아래처럼 업데이트가 동작합니다.

---

## 1. 업데이트 확인은 언제 하나요?

- **지금은**: 앱 화면에서 **「업데이트 확인」** 버튼을 **직접 눌렀을 때만** 새 버전을 확인합니다.
- **자동 확인**: 앱을 켤 때마다 자동으로 확인하는 기능은 현재 없습니다. 필요하면 **업데이트 확인** 버튼을 눌러 주세요.

---

## 2. 업데이트 확인 버튼은 어디 있나요?

- **로그인 화면** 또는 **잠금(메인) 화면** 상단 오른쪽 근처에 **「업데이트 확인」** 버튼이 있습니다.
- 이 버튼을 클릭하면, GitHub Release에서 **최신 버전**이 있는지 확인합니다.

---

## 3. 새 버전이 있으면 어떻게 되나요?

1. **확인 중** → 버튼이 "확인 중..."으로 바뀝니다.
2. **새 버전 있음** → 자동으로 **다운로드**가 시작되고, 버튼에 "다운로드 0%", "다운로드 50%"처럼 진행률이 표시됩니다.
3. **다운로드 완료** → "재시작 대기" 등으로 바뀌고, **「업데이트 다운로드 완료, 곧 재시작됩니다」** 같은 안내가 나올 수 있습니다.
4. **적용 시점**:  
   - **앱을 종료(완전히 끄기)** 하면, 그때 설치된 새 버전이 적용됩니다.  
   - 다음에 앱을 다시 실행하면 **이미 새 버전**으로 실행됩니다.
   - (바로 재시작하지 않고 잠시 쓰다가 나중에 종료해도, 종료 시점에 업데이트가 적용됩니다.)

정리하면, **「업데이트 확인」** → 자동 다운로드 → **앱 종료 시 자동 적용** → 다음 실행부터 새 버전입니다.

---

## 4. 새 버전이 없으면?

- **「현재 최신 버전입니다」** 같은 메시지가 나오고, 버튼이 잠시 후 "업데이트 확인"으로 돌아갑니다.

---

## 5. 업데이트는 어디서 받나요?

- 앱은 **GitHub Release**에서 새 버전 정보(`latest.yml`)와 설치 파일을 받습니다.
- 회사에서 **별도 업데이트 서버**를 쓰도록 설정했다면, 그 주소로 요청이 갑니다.
- 인터넷이 연결되어 있어야 업데이트 확인·다운로드가 됩니다.

---

## 요약 (이미 설치한 사람)

| 단계 | 하는 일 |
|------|----------|
| 1 | 앱을 연다. |
| 2 | **「업데이트 확인」** 버튼을 누른다. |
| 3 | 새 버전이 있으면 자동으로 다운로드된다. (버튼에 진행률 표시) |
| 4 | 다운로드가 끝나면 **앱을 한 번 종료**한다. |
| 5 | 다시 앱을 실행하면 **새 버전**으로 동작한다. |

즉, **이미 설치한 사람은 「업데이트 확인」 버튼을 누르면** 새 버전을 받고, **앱을 종료했다가 다시 켜면** 업데이트가 적용됩니다.
